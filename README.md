Sistema-de-Loja

Sistema de Gerenciamento de Estoque feito por Arthur e Rafael.

Descrição do Projeto

Estamos no terceiro ano do ensino médio e este projeto consiste no desenvolvimento de um sistema de gerenciamento de estoque utilizando Java e banco de dados. O sistema tem como finalidade cadastrar, listar e remover produtos do estoque, facilitando o controle das informações armazenadas.

Justificativa

O tema foi escolhido por ambos os integrantes devido à importância do controle de estoque no contexto comercial e à oportunidade de aplicar, de forma prática, os conhecimentos adquiridos em programação orientada a objetos e banco de dados. Além disso, trata-se de um projeto viável tecnicamente dentro do nosso nível atual de formação.

Funcionalidades Principais
Cadastro de produtos
Exclusão de produtos
Listagem de produtos cadastrados
Controle de quantidade em estoque
Armazenamento de dados em banco de dados
O sistema recebe

Dados do produto:

Nome
Quantidade
Preço
Tecnologias Utilizadas
Java
JDBC
MySQL
JOptionPane
Programação Orientada a Objetos (POO)
Objetivo

Desenvolver uma aplicação simples para auxiliar no gerenciamento de estoque, permitindo registrar e consultar produtos de forma organizada e eficiente.
